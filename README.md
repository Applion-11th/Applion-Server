## 초기 셋팅

### 1. 로컬 레포지토리 생성
```
git clone https://github.com/Applion-11th/Applion-Server.git
```

### 2. 가상 환경 생성 및 실행
- Windows
```
cd Applion-Server
python -m venv venv
source venv/Scripts/activate
pip install django
pip install -r requirements.txt
```

- Mac
```
cd Applion-Server
python3 -m venv venv
source venv/bin/activate
pip3 install django
pip3 install -r requirements.txt
```

### 1. 가상 환경 실행
